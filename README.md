# Proyecto-2---Bases-de-Datos-1

## Requisitos
- Docker

## Ejecución

1. Clonar repo
2. Crear archivo .env basado en .env.example
3. Ejecutar:

docker compose up

4. Backend disponible en:
http://localhost:3000

## Endpoints

- /ventas (JOIN)
- /clientes-con-compras (SUBQUERY)
- /ventas-mayores (GROUP BY)
- /top-clientes (CTE)
- POST /venta (TRANSACCIÓN)
